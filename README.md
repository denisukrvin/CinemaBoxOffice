# CinemaBoxOffice

## Prerequirements

* Visual Studio 2019
* .NET Core SDK
* SQL Server

## How To Run

* Open solution in Visual Studio 2019
* Change database connection string to yours in [appsettings.json](./CinemaBoxOffice.API/appsettings.json)
* In the Package Manager Console run the `update-database` command
* Run the application
